# soc.stat – Gate checklist (G0–G5)

Použi tento checklist pri každom schvaľovacom stretnutí.

## G0 Strategické ukotvenie
- [ ] Schválený one-liner produktu.
- [ ] Schválený scope MVP (must/should/later).
- [ ] Schválený tón komunikácie.

## G1 UX a etika
- [ ] Test používania: pochopenie flow do 30 sekúnd.
- [ ] Etický audit bez kritických nálezov.
- [ ] Notifikačná politika max 1/deň + opt-out.

## G2 Obsahový systém
- [ ] Content schema verzovaná.
- [ ] 30 dní obsahu pripravených.
- [ ] 95 %+ validácia bez manuálneho zásahu.

## G3 Technické MVP
- [ ] Kritická cesta E2E bez blokera.
- [ ] Monitoring a incident runbook aktívny.
- [ ] Error rate/uptime v limite.

## G4 Monetizácia
- [ ] Premium cenník aktívny.
- [ ] Gold pravidlá transparentné.
- [ ] Žiadny signál pay-to-win.

## G5 Stabilizácia a compliance
- [ ] GDPR checklist PASS.
- [ ] Security baseline PASS.
- [ ] Pripravený plán extrakcie reusable modulov.
